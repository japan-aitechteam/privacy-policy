プライバシーポリシー

本アプリ（以下「当アプリ」）は、利用者のプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーでは、当アプリにおける利用者の情報の取り扱いについて説明します。

1. 取得する情報

当アプリでは、以下の情報を取得する場合があります。

  • ユーザーが撮影した食材等の写真（画像内に個人識別情報が含まれる可能性がある場合も含みます）
  • 広告配信のための端末識別情報（Google AdMob 等）
  • アプリの利用状況に関する情報（Google Analytics 等）
  • API キー取得のための外部サーバーとの通信ログ

2. 利用目的

取得した情報は、以下の目的で使用されます。

  • ChatGPT API（OpenAI）を利用した画像認識処理（例：食材のリスト化）
  • 認識結果をユーザーに返すため
  • アプリの利便性・品質向上のため
  • 広告の最適化・パーソナライズのため
  • アプリの不具合調査や利用状況の分析のため
  • 外部APIキー取得のための認証処理

3. 第三者提供

取得した情報は、法令に基づく場合を除き、第三者に提供することはありません。

ただし、広告配信や利用状況解析のため、以下の第三者サービスに情報が送信される場合があります。これらは当該サービス提供者のプライバシーポリシーに基づいて取り扱われます。

  • Google AdMob プライバシーポリシー
  • Google Analytics プライバシーポリシー

4. 利用している外部サービス

当アプリでは、以下の外部サービスを利用しています。

  • Google AdMob（広告配信）
  • Google Analytics for Firebase（利用状況解析）
  • OpenAI ChatGPT API（画像認識処理）
  • 当アプリ提供元が管理する外部サーバー（APIキー取得）

なお、ChatGPT API（OpenAI）に送信される画像データは、OpenAIのポリシーに従い、処理目的のみに使用され、モデルの学習や改善には使用されません。処理後は適切に削除され、サーバーに長期間保存されることはありません。

参考：OpenAI APIのデータ利用ポリシー

5. 情報の管理

取得した情報は、通信の暗号化、一時ファイルの速やかな削除、外部ストレージへの保存制限など、適切な安全対策を講じた上で管理し、漏洩・改ざん・不正アクセスなどの防止に努めます。

また、当アプリでは、ユーザーの画像データをアプリ内で一時的に処理するのみであり、サーバー等への保存は行いません。
