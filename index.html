プライバシーポリシー

本アプリでは、ユーザーのプライバシーを尊重し、個人情報の保護に最大限配慮しております。以下に、当アプリで取得・利用される情報とその取り扱いについてご説明いたします。

1. 取得する情報

本アプリでは、以下の情報を取得します。

  • ユーザーが撮影した写真

2. 情報の利用目的

取得した写真は、以下の目的にのみ使用されます。

  • ChatGPT API（OpenAI）を利用した画像認識処理
  • 認識された内容のリスト化および結果をユーザーに返す処理

3. 情報の第三者提供

取得した写真は、ChatGPT API を通じて画像認識処理のために一時的に送信されますが、OpenAI による学習目的での保存や再利用は行われていません（※OpenAIの利用規約およびプライバシーポリシーに準拠）。

また、ユーザーの情報をその他の第三者に提供することはありません。

4. データの保存

撮影された写真はアプリ内処理のために一時的に使用されます。

画像データや認識結果は、当アプリによってサーバー上に保存されることはなく、ユーザーのスマートフォン上で完結します。
